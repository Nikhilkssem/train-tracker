# 🚆 Train Tracker (Where Is My Train)

A Django-based web application to track train status, location, and delay.

## 🔥 Features
- Train search by number
- Live train status (AJAX)
- Delay prediction
- Interactive charts (Chart.js)
- Map integration
- Mobile-friendly UI

## 🛠 Tech Stack
- Django
- HTML, CSS, Bootstrap
- JavaScript (AJAX)
- Chart.js

## ▶️ Run Project
python manage.py runserver
